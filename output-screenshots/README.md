# Output Screenshots

This folder contains screenshots showcasing the execution and results of the Maze Solver using AI Search Algorithms project.

# Contents

The screenshots included here demonstrate:

- Generated mazes of different difficulty levels)<br>
- Execution of various search algorithms (BFS, DFS, Best First, A*))<br>
- Pathfinding results from start (S) to end (E))<br>
- Colored visualization of explored paths and final solution)<br>
- Performance comparison outputs (time taken and steps)<br>

# Purpose

These screenshots provide a visual understanding of how the algorithms work and how their performance differs under similar conditions.
